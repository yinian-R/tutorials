# NACOS
一个更易于构建云原生应用的动态服务发现、配置管理和服务管理平台。

## 引入依赖
```
<dependency>
    <groupId>com.alibaba.nacos</groupId>
    <artifactId>nacos-client</artifactId>
    <version>2.0.3</version>
</dependency>
```
## 新建命名空间
访问 http://127.0.0.1:8848 -> 命名空间，新建命名空间
