它们中具有相同配置时，优先级是怎么的，最终会以那个为准
```
sh 命令行 --spring.config.location=../conf/bootstrap.yml

bootstrap.yml 里面追加的配置 spring.config.additional-location 添加的配置

nacos.config 中的 shared-configs、extension-configs
```

**综合优先级**
```
按照优先级从低到高排序：
默认配置文件（如 application.properties 或 application.yml）。
--spring.config.location 指定的配置文件。
spring.config.additional-location 指定的配置文件。
环境变量或系统属性。
命令行参数。
Nacos 动态配置（如果启用了动态刷新）。
```
